# Feature Inventory

This document contains the complete feature inventory for Seamdoc.

Features are grouped into logical modules and prioritized for implementation.

Priority Levels:

- **P0** — Required for MVP
- **P1** — Planned after MVP
- **P2** — Future Vision

# MVP Scope (v0.1)

The first public release includes only the following modules:

- Markdown Editor
- Live Preview
- DOCX Generation
- Theme Engine
- Document Settings
- Performance
- Accessibility
- Application Settings

Everything else is intentionally postponed to later milestones.
